# 1.1 Frontmatter Parser [DONE]

## Description
Parse YAML frontmatter from card files. Frontmatter is delimited by `---` lines
at the start of the file. Supports scalar values (`parent: path`) and list values
(`children:\n  - path`).

## File Manifest
- src/card.ts

## Acceptance Criteria
- Parses scalar key-value pairs
- Parses list values (indented `- item` lines)
- Returns empty object for files without frontmatter
- Handles mixed scalar and list values
- stripFrontmatter() removes the frontmatter block from content
