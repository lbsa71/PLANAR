# 1.4 GUID Conversion [DONE]

## Description
Deterministic dot-path ↔ GUID conversion. Each path segment maps to a two-byte
hex pair, zero-padded into the standard GUID layout. Used as session IDs for
Claude Code CLI invocations.

## File Manifest
- src/guid.ts

## Acceptance Criteria
- Single segment: "1" → "01000000-0000-0000-0000-000000000000"
- Multi-segment: "2.3.1" → "02030100-0000-0000-0000-000000000000"
- Round-trips: guidToDotPath(dotPathToGuid(x)) === x
- Rejects segments outside 0-255 range
- Rejects all-zero GUIDs
