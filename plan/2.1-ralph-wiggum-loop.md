# 2.1 Ralph Wiggum Loop [DONE]

## Description
Core iteration loop: read card, check status, invoke Claude Code CLI via
ClaudeInvoker interface, diff before/after, detect convergence. Uses
execFileSync with args array (not shell string) to avoid injection.

See @README.md — "The Ralph Wiggum Pattern" and "Termination" sections.

## File Manifest
- src/wrapper.ts
- src/types.ts

## Acceptance Criteria
- Exits on DONE status
- Exits on special statuses (BLOCKED-BY, CONFLICTS-WITH, INACTIONABLE)
- Detects convergence (card unchanged after invocation)
- Respects maxIterations cap
- Tracks accumulated cost and exits on budget exhaustion
- Aborts on link integrity errors before invocation
- Warns on link integrity errors after invocation
- ClaudeInvoker injected for testability
