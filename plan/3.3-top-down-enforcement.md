# 3.3 Top-Down Enforcement [DONE]

## Description
Enforce the top-down wavefront: agents cannot be spawned for cards whose ancestor
nodes are still being decomposed (in PLAN phase) or have active agents.

See @README.md — "Top-Down Resolution" section.

## File Manifest
- src/orchestrator.ts

## Acceptance Criteria
- Blocks child when parent node is in PLAN
- Blocks child when any ancestor has an active agent
- Blocks deeply nested cards when grandparent is in PLAN
- Allows child when parent node is DONE
- Excludes cards with special statuses (BLOCKED-BY, CONFLICTS-WITH, INACTIONABLE)
- Excludes cards that exceeded max iterations
- Respects blocked-by dependencies (blocks until dependency is DONE)
