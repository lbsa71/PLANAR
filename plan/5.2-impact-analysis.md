# 5.2 Impact Analysis & Card Creation [PLAN]

## Description
Cross-reference changed files against every card's file manifest to find
affected cards. Create an impact card (plan/0.N-upstream-sync.md) documenting
the commit range, changed files, and affected cards with a diffstat summary.

See @README.md — "Git Watch Mode: How It Works" steps 5-6.

## Acceptance Criteria
- Matches changed files against all card file manifests
- Creates impact card with YAML frontmatter and proper structure
- Impact card includes: commit range, file list, affected card list, diffstat
- Impact card starts in [PLAN] phase
- Watcher generates diffstat mechanically (no LLM reasoning)
- Sequential impact cards get incrementing dot-paths (0.1, 0.2, ...)
